## Module Summaries

### Task 1 – Digital Literacy Infographic
Created a one-page visual infographic covering four topics: what digital literacy is, useful digital tools for students, safe internet practices, and building a professional online presence. Designed using Python (Pillow library).

### Task 2 – Student Digital Portfolio
Set up professional profiles on **GitHub**, **LinkedIn**, and **HackerRank**. Created a GitHub profile README with name, branch, year, and learning goals. Filled in LinkedIn Education section.

### Task 3 – Coding & Collaboration Platforms
Completed a beginner-level coding challenge on **HackerRank** (Solve Me First). Built a **Google Form** Digital Literacy Awareness Quiz with 5 questions linked to a Google Sheet for response tracking.

**Google Form Quiz Link:** _(Paste your Google Form link here)_

### Task 4 – Email Etiquette
Drafted two professional emails: (1) requesting an assignment extension from a professor, and (2) expressing interest in a summer internship. Also created a Social Media Do's and Don'ts checklist with 6 Do's and 6 Don'ts.

### Task 5 – Cybercrime Awareness
Wrote a 200+ word case study on UPI/Online Payment Fraud in India, including step-by-step breakdown of how the crime occurs. Created a 10-point "Stay Safe Online" prevention checklist with UPI-specific safety tips and reporting channels (cybercrime.gov.in | Helpline 1930).
