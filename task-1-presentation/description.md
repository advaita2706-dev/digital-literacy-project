# Task 1 – Digital Literacy Awareness Infographic

**Tool Used:** Python (Pillow library) – a free, browser-accessible image-creation tool  
**File:** `digital_literacy_infographic.png`

## What the Design Shows

The infographic is a single-page visual resource divided into four colour-coded sections:

1. **What is Digital Literacy?** – Defines the concept in plain language, covering information evaluation, online safety, digital footprint awareness, and critical thinking about online content.
2. **Digital Tools for Students** – Highlights free, practical platforms (Google Workspace, GitHub, Canva, HackerRank, Notion) that every engineering student should know.
3. **Safe Internet Practices** – Lists five actionable habits: strong passwords with 2FA, link verification, never sharing OTPs, keeping software updated, and avoiding public Wi-Fi for sensitive tasks.
4. **Professional Online Presence** – Explains the importance of LinkedIn, GitHub portfolios, responsible social media use, and professional email communication.

Each section uses a distinct accent colour for quick visual scanning. A header banner and footer bar frame the layout with course and institution details.

The design was created entirely with free tools, requiring no subscription or design experience — reflecting the spirit of digital literacy itself.
