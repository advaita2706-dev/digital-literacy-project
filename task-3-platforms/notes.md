## Reflection (150–200 words)

For the coding practice portion of this task, I used **HackerRank** to complete a beginner-level problem-solving challenge. The platform provides an in-browser code editor with automatic test-case evaluation, which makes it extremely beginner-friendly — I did not need to set up any development environment on my own machine. Seeing my solution pass all test cases on the first attempt was genuinely motivating and made me want to tackle harder problems.

For the collaboration portion, I used **Google Forms** — a free tool that is part of Google Workspace — to build a five-question Digital Literacy Awareness Quiz. What impressed me was how seamlessly Forms links to Google Sheets: every response is automatically recorded in a spreadsheet, which means the data is ready for analysis without any manual work.

Academically, both tools are highly valuable. HackerRank will prepare me for coding assessments during campus placements and technical interviews. Google Forms will be useful for collecting survey data in research projects, club activities, and peer assessments. Together, they represent the kind of practical digital skills that are expected of engineering graduates today.
